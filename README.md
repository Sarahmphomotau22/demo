# Footprint Logger Platform

## Overview
A web application that allows users to log daily activities contributing to carbon emissions, view summaries, and receive personalized tips to reduce their footprint.

## Features
- Activity logging with CO₂ values
- Visual summaries (pie chart)
- User accounts with registration/login
- Dashboard with totals and community comparison
- Insight engine for personalized tips

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB
- Persistence: LocalStorage (single-session)
